## Запуск проекта:

Устанавливаем необходимые зависимости.

`sudo apt-get install php7.*-mysql php7.*-pdo`

Поднимаем контейнер

`docker-compose up`

Создаем базу данных

`sudo chmod +x bin/start.sh`

`bin/start.sh`

Заходим в браузере

`http://127.0.0.1:8000/`
